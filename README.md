**오늘의 켠왕(켠김에 왕까지) 3선을 소개하는 웹 페이지**입니다.  

LoL / 얼불춤 / 산나비 세 가지 게임을 중심으로  
각 게임에 대한 소개, 배경 음악, 인터랙션을 구현했습니다.

---

## 📌 프로젝트 개요

- **프로젝트 이름**: Today Speedrun King Top3
- **목적**
  - 팀이 좋아하는 게임을 주제로 한 “켠왕” 콘셉트의 웹 페이지 제작
  - HTML, CSS, JavaScript 기본기 및 협업 연습
  - 스크롤, 오디오, 비디오, 인터랙션 등을 직접 구현해보는 것에 중점

---

## 📌 직접 구현한 기능들 (요약)

1. **상단 스크롤 진행바(UI)**  
2. **게임별 섹션 구조 및 레이아웃 구성**  
3. **각 게임의 설명 박스**  
4. **게임별 영상(mp4) 재생 영역**  
5. **스크롤 위치에 따라 자동으로 바뀌는 배경 음악 로직(섹션 기반 BGM)**  
6. **고정된(플로팅) 오디오 플레이어 – 현재 섹션 음악만 표시**  
7. **퀴즈/참여형 요소(오늘의 켠왕 관련 질문 섹션)**   
8. **인터랙션 & 스타일 상세**

---

## 1. 상단 스크롤 진행바

- 화면 최상단에 `div#scroll-progress`를 추가하여 구현.
- JavaScript로 `window.scrollY`와 전체 문서 높이를 계산해,
  현재 스크롤 비율에 따라 `width`를 동적으로 변경.
- 사용자가 페이지를 얼마나 내려왔는지 한눈에 볼 수 있는 **진행 상태 표시바 역할**.

---

## 2. 게임별 섹션 구조 및 레이아웃

- HTML에서 각 게임을 **별도 섹션**으로 분리 (예: `section id="lol"`, `id="adofai"`, `id="sanabi"` 형태).
- 각 섹션에는 다음 요소들이 공통적으로 포함되도록 설계:
  - 섹션 제목 (게임 이름)
  - 간단한 게임 소개 문단
  - 대표 이미지(썸네일 역할)
  - Top3 콘텐츠/설명 카드
- CSS Grid / Flexbox를 사용해 **가로 카드 배치, 세로 정렬, 중앙 정렬** 등을 조정.

---

## 3. 각 게임의 설명 박스

- 각 게임 섹션에서 게임의 설명을 보여주는 레이아웃 구현.
  - 각 카드에 다음 정보 배치:
    - 제목 또는 간단한 설명
    - 어떤 플레이/영상인지 설명하는 텍스트
---

## 4. 게임별 영상(mp4) 재생 영역

- 저장소에 `lol.mp4`, `adofai.mp4`, `sanabi.mp4` 등 게임별 영상 파일을 추가.
- 각 섹션에 `<video>` 태그를 두고 해당 mp4를 연결하여,
  사용자가 페이지 안에서 직접 영상을 볼 수 있도록 구현.

---

## 5. 섹션 기반 배경 음악 자동 전환

> “스크롤해서 롤 쪽에 가면 롤 BGM, 얼불춤은 얼불춤 BGM, 산나비는 산나비 BGM 나오게”

이 요구사항을 만족시키기 위해 다음과 같이 구현:

- 각 게임별로 **별도의 오디오 요소**를 준비 (예: `lol_bgm.mp4`, `adofai_bgm.mp4`, `sanabi_bgm.mp4`).
- JavaScript에서 **현재 보고 있는 섹션을 감지**:
  - `IntersectionObserver` 또는 `scroll` 이벤트를 이용해
  - 사용자의 스크롤 위치와 각 섹션의 위치를 비교
- 현재 섹션이 화면의 기준 영역에 들어오면:
  - 해당 섹션의 배경 음악을 `play()`
  - 다른 섹션 음악들은 `pause()` 또는 `currentTime = 0` 등으로 정지
- 한 번에 하나의 음악만 재생되도록 제어하여  
  섹션과 음악이 자연스럽게 매칭되도록 설계.

---

## 6. 플로팅 오디오 플레이어(섹션에 따라 표시 변경)

- 화면 옆(사이드)에 **작은 음악 플레이어 박스**를 두고,
  현재 재생 중인 게임의 정보를 표시.
  - 예: `Now Playing: League of Legends BGM`
- 스크롤에 따라 섹션이 바뀔 때:
  - 플레이어의 제목/텍스트도 함께 변경되도록 JS로 연결
- 사용자는:
  - 배경 음악 재생 / 일시정지 버튼
  를 통해 배경음악을 직접 제어 가능.

---

## 7. 켠왕 퀴즈 / 참여형 요소

- **간단한 퀴즈/질문 섹션**을 추가.
- 버튼 클릭형 퀴즈 형태:
  - 선지 버튼을 누르면 정답/오답 문구를 아래에 표시
  - 간단한 설명(해설)을 함께 보여주어 **학습 + 재미 요소** 추가
- JavaScript로 버튼 클릭 이벤트를 처리하여,
  - 특정 `div` 영역에 결과 텍스트를 출력하는 방식으로 구현.

---

## 8. 인터랙션 & 스타일 상세

- 전체 페이지에 공통으로 적용한 CSS:
  - 어두운 배경 + 네온 느낌의 포인트 컬러
  - 제목에 `text-shadow`를 활용해 **게임 UI 같은 분위기** 연출
- 카드, 버튼, 이미지 등에 공통 적용:
  - `transition`으로 부드러운 hover 효과
  - `transform: scale()` 등을 사용한 살짝 확대 애니메이션
- 섹션 간 여백, 구분선, 타이포그래피 등을 조정해
  정보가 너무 답답하게 보이지 않도록 레이아웃 설계.

## 📌 실행 방법

가장 간단한 사용 방법:

1. 이 저장소를 **클론 또는 ZIP 다운로드** 합니다.
2. 압축을 풀고, 폴더 안의 `index.html` 파일을 더블 클릭합니다.
3. 크롬 등의 브라우저에서 페이지가 열리며,
   - 스크롤에 따라 섹션 이동  
   - 구간별 배경 음악 재생  
   - 동영상 재생 및 퀴즈 기능  
   을 확인할 수 있습니다.
